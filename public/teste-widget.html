Teste isolado do Widget AppyPay

Esta página não depende do site Zango nem do Supabase — serve só para confirmar se as credenciais (ClientID + ApiKey) funcionam diretamente com o widget oficial da AppyPay. Se o widget carregar e mostrar o ecrã de pagamento, as credenciais estão OK. Se der erro de autenticação, o problema é mesmo nas credenciais / conta.

Nota: se o quadro acima ficar em branco, abre a consola do browser (Definições → Avançado → Web Inspector no Safari, ou F12 no desktop) e vê se aparece algum erro de rede ou de autenticação — copia esse erro e envia.
